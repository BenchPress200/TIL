- 리트코드<br>
**[[Equal Sum Grid Partition 1](https://leetcode.com/problems/equal-sum-grid-partition-i/description/) - 19:33]**<br>
    - 양의 정수가 담긴 m * n 크기 2차원 배열 주어짐
    - 1 <= m * n <= 10^5
    - 1 <= 격자 한 칸 정수 <= 10^5
    - 행 또는 열을 하나 정해서 전체 격자를 둘로 나눴을 때, 각 영역의 합이 서로 같다면 true 반환, 아니라면 false 반환

    입력받은 격자의 최대 칸수는 100,000이기 때문에 행 또는 열을 정해서 그때마다 단순히 합을 구해서 비교하면 시간초과가 예상되었습니다. 그래서, 격자의 각 정수를 누적합 배열로 저장하고 행 또는 열을 선택해 풀이했습니다. 여기서 격자 모든 값을 더했을 때 int 범위를 넘어설 수 있으므로 누적합 배열의 타입을 long 선언해야만 합니다.
